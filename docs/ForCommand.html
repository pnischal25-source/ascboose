Class ForCommand
Represents a BOOSE FOR loop command. Executes a block of commands repeatedly while updating a loop variable from a start value to an end value using a step.
Syntax
C#
    public class ForCommand : IBOOSECommand

Documentation Preview
Constructor ForCommand
Initializes a new instance of the ForCommand class.
Syntax
C#
    public ForCommand(string variable, double start, double end, double step, BlockCommand block)
Parameters
variable
Name of the loop variable.
start
Initial value of the loop variable.
end
Final value at which the loop terminates.
step
Amount by which the loop variable is incremented or decremented.
block
Block of commands executed during each iteration.
