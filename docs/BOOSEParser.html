Class BOOSEParser
Parses BOOSE program source code into executable command objects. This parser supports:
  • Simple drawing commands
  • Variables and arrays
  • Conditional logic (if / else)
  • Loop constructs (while / for)
  • User-defined methods

Method ParseLines
Parses a sequence of lines into executable commands until an optional end marker is encountered. This method is used recursively for blocks such as:
  • method bodies
  • if / else blocks
  • while loops
  • for loops
Syntax
C#
    private IList<IBOOSECommand> ParseLines(string[] lines, ref int index, params string[] endMarkers)
Parameters
lines
All program source lines.
index
Current parsing position (passed by reference).
endMarkers
Optional keywords that terminate the current block.
Return Value
A list of parsed commands within the block.
