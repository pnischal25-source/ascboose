Interface IBOOSECommand
Defines the structure of a BOOSE command that can be executed by the interpreter. All BOOSE commands (e.g., moveto, drawto, circle, rect, write) implement this interface to provide a consistent execution model.
Syntax
C#
    public interface IBOOSECommand

Documentation Preview
Method Execute
Executes the command using the provided BOOSEContext. Each command implements its own behavior, such as drawing shapes, moving the pen, changing colour, or writing text.
Syntax
C#
    void Execute(BOOSEContext context)
Parameters
context
The current interpreter context, holding drawing tools, canvas, pen state, and pen position.
